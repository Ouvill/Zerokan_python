* これは何？
  WarThunder の戦績を記録するソフトです。試合が終了するとdata.csvに戦績が書き込まれます。また、試合終了後にTwitterに戦績を投稿する機能もあります。
  記録されるデータ
  試合開始時間、試合終了時間、撃墜数、被撃墜数、墜落数、地上物破壊数、戦車を破壊された数、戦車で自滅した数
  バグが大量にあるかと予想されます。それでも大丈夫ってひとだけ使ってください。

* 初回設定
  - setting.ini を開いて、初期設定を入力してください。

* 使い方
  - WT起動中(ランチャー画面、格納庫画面時)にzerokan.exeを起動してください。

  - ゲームが終了するごとにdata.csvに試合結果が記録されます。
    Twitter機能を有効にしている場合、試合結果がツイートされます。

* Twitter 機能の有効の仕方
  ツイッターのキーを取得する方法を実装できていないので、皆さん各自でキーを取得してください。
  取得の仕方は "Twitter機能有効化の方法.html" を読んでください。
  正直、現段階で利用することはお勧めしません。

* ファイル構成
  - readme.txt このファイル
  - Manual Twitter 機能の設定方法について
  - zerokan.exe プログラム本体
  - data.csv 戦績の記録
  - setting.ini 設定ﾌｧｲﾙ

* アンインストール方法
  WT_Flight_Recorderフォルダを全て削除してください。

* 既知のバグ
  - 味方を撃墜しても、killのカウントになります。
  - その他、沢山
